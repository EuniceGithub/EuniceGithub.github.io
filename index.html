this关键字

  • this： this 表示包含它的函数作为方法被调用时对所属对象的一个引用。

  • this的指向：

    JavaScript 中 this 不是固定不变的，它会随着执行环境的改变而改变，实际上 this的最终指向的是那个调用它的对象。

    一般 this 的指向有这几种情况

    1. 全局环境下的 this

      全局环境下单独使用 ，this 指向全局对象。

      在浏览器中，window 就是该全局对象。

    2. 函数中使用this

      • 默认：

        在函数中，函数的所属者默认绑定到 this 上。

        在浏览器中，window 就是该全局对象。
